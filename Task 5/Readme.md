# 📊 Task 5: Interactive Business Dashboard in Streamlit

## 📌 Project Overview

This project is an interactive Business Intelligence dashboard built using Streamlit and the Global Superstore Dataset. The dashboard provides a clear overview of sales, profit, customer performance, and segment-wise business insights through interactive filters and visual KPI analysis.

The goal of this project is to transform raw sales data into meaningful visual insights that can support business decision-making.

---

## 🎯 Task Objective

The objective of this task is to develop an interactive dashboard for analyzing sales, profit, and segment-wise performance.

Key objectives include:

* Clean and prepare the Global Superstore dataset
* Build an interactive Streamlit dashboard
* Add filters for Region, Category, and Sub-Category
* Display key performance indicators
* Visualize sales, profit, customer, and segment performance
* Enable data storytelling through interactive charts

---

## 📂 Dataset

**Dataset:** Global Superstore Dataset

The dataset contains sales transaction details from a global retail business.

### Important Columns Used

* Order Date
* Region
* Category
* Sub-Category
* Segment
* Customer Name
* Sales
* Profit
* Quantity
* Order ID

## 🔍 Dashboard Features

### Interactive Filters

The dashboard includes sidebar filters for:

## 🚀 How to Run the Dashboard

### 1. Install Required Libraries

```bash
pip install streamlit pandas plotly
```

### 2. Keep the Dataset in the Same Folder

Make sure the dataset file is named:

```text
superstore.csv
```

### 3. Run the Streamlit App

```bash
streamlit run "Interactive Business Dashboard in Streamlit.py"
```
